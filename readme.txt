Nombre : Paula Averza

Proyecto: Pre-entrega nro 3


Modelos:
Discografia (nombre, año)
Integrante (nombre, apellido, profesiona)
Fecha (lugar, dia_mes)



Funcionalidad: Este proyecto es de una pagina web de una banda musical, con la posibilidad de ver la discografía y cargar nuevos discos, ver las próximas fechas de presentación, conocer a los integrantes y contactar a la banda mediante redes sociales.
